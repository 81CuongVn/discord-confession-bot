# discord-confession-bot

## Cách cài đặt

### Docker
- Cần tải Docker và docker-compose
- Clone repo về
- Đổi tên `example.env` thành `.env`
- Thay đổi `TOKEN` bằng Token của bot
- Xoá dòng `MONGODB`
- `docker-compose up -d` để chạy container

### Chạy trực tiếp
- Cần tải nodejs
- Clone repo về
- `npm i`
- Đổi tên `example.env` thành `.env`
- Thay đổi `TOKEN` bằng token của bot
- Thay đổi `MONGODB` bằng link tới mongodb
- `npm start`
